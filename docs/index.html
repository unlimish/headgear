気圧の変化を予測し、
体調管理をスマートに。

A Discord bot fetching real-time atmospheric pressure forecasts in Japan from Zutool. Keep track of pressure changes to predict weather-related headaches.

頭痛や気象病の原因となる気圧の変化を、Discord上で素早く確認できるボットです。ずつーるのデータをもとにリアルタイムで気圧予報を取得します。

Pressure Forecasts

気圧予報の取得

Fetch real-time atmospheric pressure trends and warnings for municipalities across Japan directly using Zutool data.

全国の市区町村を対象に、リアルタイムの気圧予報や注意レベル（警戒、爆弾など）をシンプルかつ素早く取得します。

Personal Install

マイアプリ対応

Support for Discord User Installable App. Add to your personal profile to check forecast inside any DM or guild without server installation.

Discordの「マイアプリ（個人アカウントへの追加）」に対応。サーバー管理者でなくても、任意のDMやギルド内でコマンドを実行できます。

Auto Presence

ステータスの自動更新

Updates client presence status automatically every 15 minutes to reflect the current local atmospheric pressure and warning levels.

ボットのアカウントステータス（オンライン・カスタムステータス）が15分おきに自動更新され、登録地域の直近の気圧をひと目で確認できます。

Commands

コマンド・連携機能

/zu2 [city_name] [date] Retrieve pressure forecasts with interactive date toggle buttons. 指定した都市の気圧予報を表示します。ボタン切り替えで昨日から明後日までの予報を確認できます。
Slash Command スラッシュコマンド
/zu2set [city_name] Save your default city for commands. Clear configuration by passing no arguments. 自分のデフォルト都市を保存します。引数なしで実行すると登録をクリアします。
Slash Command スラッシュコマンド
Show Pressure (気圧の表示) Right-click on any user -> Apps to fetch forecast for their registered city. ユーザーを右クリック ＞ アプリ から実行し、そのユーザーが登録している都市の気圧予報を表示します。
